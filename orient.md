# Orient: WCSD AI Hub

## What This Is

A single-file prototype website for the WCSD AI Hub: the district's teacher-facing AI resource. Landing page plus five functional sections (Prompt Library, Tool Guides, Guardrails, Tips, Summer Cohort). Built 2026-06-07 for JT's leaders demo (Mon 6/8) and the teacher cohort kickoff (Wed). It is the third member of the Walk + NSPF Calculator product family and uses their exact design language.

## Architecture Decisions (Locked)

1. **Single-file HTML, vanilla JS, zero frameworks, no CDN dependencies.** Same pattern as NSPF v17. Offline-safe for demos.
2. **NSPF design discipline:** navy/gold tokens only, white topnav, squared 6-8px tags, no purple/teal/#E8B82E. Stylesheet is derived from `output/nspf-calculator/toolchain/nspf-huddle.css` (the family source of truth) plus BRAND.md tokens.
3. **Prompt bodies are JT's published doworkwithai IP, kept byte-faithful.** They are inside the `const PROMPTS = [...]` array. The em-dash ban and voice rules apply to UI copy only, never to the prompt bodies.
4. **Pretext evaluated, not used:** static content site, native browser text layout is sufficient (per CLAUDE.md "when NOT to use").
